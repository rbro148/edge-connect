
This is the irregular mask dataset.

The black and gray pixels indicate the holes. During the training, we use random rotation, dilation, cropping etc to augment these masks.

 * This dataset is for research purposed only. *


Reference: 

Image Inpainting for Irregular Holes Using Partial Convolutions 
Guilin Liu, Fitsum A. Reda, Kevin J. Shih, Ting-Chun Wang, Andrew Tao, Bryan Catanzaro
ECCV 2018
